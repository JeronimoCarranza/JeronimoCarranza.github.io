Migración de herramientas hidrológicas en entorno SIG.

 

Posición: Jefe de proyecto.
Destinatarios: Confederación Hidrográfica del Guadiana. 2022.

Los trabajos realizados corresponde a la migración de diversas herramientas hidro-meteorológicas de las que dispone la Confederación Hidrográfica del Guadiana,desarrolladas para ArcGIS, al Sistema de Información Geográfica abierto QGIS, utilizando el lenguaje de programación Python. Concretamente se han desarrollado los siguientes scripts de geoprocesamiento:

  • Análisis de cuenca vertiente. Selección de masas (o subcuencas) aguas arriba/abajo.
  • Análisis sentido hidrológico. Selección de tramos de cursos de agua, aguas arriba/abajo.
  • Análisis de cuenca de aportación. Cálculo de cuencas de aportación en puntos seleccionados.
  • Extracción de datos SIMPA. Dada una capa de polígonos se realiza el cálculo de valores acumulados para los diferentes recintos a partir de las variables del modelo SIMPA del CEDEX. La aplicación permite seleccionar el rango de años hidrológicos (1940-2018) para los que se realiza el cálculo. El resultado es un conjunto de tablas (no geométricas) que se almacenan en un fichero geopackage (gpkg).
